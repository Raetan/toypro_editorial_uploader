# No.018 来店人数 - 300

この問題は「lの中にいる人の種類数はいくつですか？」という問題に置き換えることができます。
pythonでは
```py
set(l)
```
と書くことによって重複した要素を取り除いた配列を作ることができます。
また、配列の長さは
```py
print(len(l))
```
とすることによって出力することができます。
### 実装例
```py
l = ['a', 'b', 'b', 'b', 'c', 'b', 'e', 'd', 'c', 'a', 'f', 'w', 'w', 'q', 'm', 'm', 'a']

n = set(l)

print(len(n))
```
